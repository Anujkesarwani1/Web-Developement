What is HTML?

  • HTML stands for Hyper Text Markup Language
  • HTML is the standard markup language for creating Web pages
  • HTML describes the structure of a Web page
  • HTML consists of a series of elements
  • HTML elements tell the browser how to display the content
  • HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

A Simple HTML Document

  <!DOCTYPE html>
  <html>
      <head>
          <title>Page Title</title>
      </head>
      <body>
          <h1>My First Heading</h1>
          <p>My first paragraph.</p>
      </body>
  </html>

Example Explained

  • The <!DOCTYPE html> declaration defines that this document is an HTML5 document
  • The <html> element is the root element of an HTML page
  • The <head> element contains meta information about the HTML page
  • The <title> element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab)
  • The <body> element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
  • The <h1> element defines a large heading
  • The <p> element defines a paragraph

Comments

  • Used a add notes in HTML code
  • Not displayed on the web page
  • Syntax: <!--Comment here -->
  • Helpful for code organization
  • Can be multi-line or single-line

Case Sensitivity

  • HTML is case-insensitive for tag names
  • Attributes names are also be case-insensitive
  • Best practice: use lowercase for consistency

HTML Attributes

  • Provides additional information about elements
  • Placed within opening tags
  • Common examples: href, src, alt
  • Use name=value format
  • Can be single or multiple per element

Some must used tags

Heading Tag
  • Defines headings in a document
  • Ranges from <h1> to <h6>
  • <h1> is most important, <h6> is least
  • Important for SEO
  • Helps in Structuring content
Paragraph Tag
  • Used for defininig paragraphs
  • Enclosed within <p> and </p>
  • Adds automtic spacing before and after
  • Text wraps to next line inside tag
  • Common in text-heavy content
<BR> Tag
  • <br> adds a line break within text
  • <br> is empty, no closing tag needed
  • <br> and </br> both valid
<HR> Tag
  • <hr> creates a horizontal rule or line
  • <hr> also empty, acts as divider
Image Tag
  • Used to embed images
  • Utilizes the src attributes for image URL
  • alt attribute for alternative text
  • Can be resized using width and height
  • Self-closing, doesn't require an end tag
Video Tag
  • Embed video files on a pages
  • Usese src attribute for video URL
  • Supports multiple formats like MP4, WebM
  • Allows for built-in controls via attributes like autoplay, controls, loop
Bold / Italic / Underline / Strikethrough Tag
  • <b> makes text bold
  • <i> makes text italic
  • <u> underlines text
  • <s> or <strike> applies strikethrough
  • Primary used for text styling and emphasis
Pre Tag
  • Preserves text formatting
  • Maintains whitespaces and line breaks
  • Useful for displaying code
  • Enclosed within <pre> and </pre> tags
Big / Small Tag
  • <big> increases text size
  • <small> decreases text size
  • Less common due to CSS alternatives
Superscript / Subscript Tag
  • <sup> makes text superscript
  • <sub> makes text subscript
  • Used for mathematical equations, footnotes
  • Does not change font size, just position


Semantic vs Non-Semantic
Semantic Tags Non-Semantic Tags
Meaningful: Describe content. Generic: No specific meaning.
SEO: Good for search engines. For Styling: Used for layout.
Accessibility: Useful for screen readers. No SEO: Not SEO-friendly.
